## API接口 错误代码

| 代码 | 解释                              |
| ---- | --------------------------------- |
| 0    | 操作成功                          |
| 1    | 内部服务器错误                    |
| 2    | 请求的参数错误                    |
| 3    | 给定的 token 无效（用于一般接口） |
| 4    | 权限不足                          |
| 5    | 找不到用户                        |
| 6    | 用户已被禁用                      |
| 7    | 给定的凭据无效（用于 /session）   |
| 8    | 重复创建活动                      |
| 9    | 找不到这个活动                    |
| 10   | 需要实名认证                      |
| 11   | 图片存在敏感内容                  |
| 12   | 文字存在敏感内容                  |
| 13   | 活动未报名                        |
| 14   | 晚安计划未报名                    |
