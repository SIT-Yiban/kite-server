## 通知

通知信息用于小程序首页显示的Toast、用于宣传小程序新版本新功能或是某些紧急通知。

接口 | 说明
---- | ----
`GET /notice` | 获取通知

## 接口

### GET /notice

获取数据库中当前显示的通知，后端返回时已排序。排序权重：置顶状态 (`top`)，时间 (`publish_time`)

#### 权限

所有用户

#### 响应示例

``` json
[
  {
    "id": 1,
    "publishTime": "2020-09-16T08:53:34.944747",
    "title": "上海多雨，记得带伞哦",
    "content": null,
    "top": true
  },
  {
    "id": 2,
    "publishTime": "2020-09-16T08:53:34.944747",
    "title": "一食堂好吃，二食堂三楼好吃，三食堂最好吃！",
    "content": null,
    "top": false
  }
]
```
